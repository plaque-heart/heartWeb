This is the core of the project and it is made using Unity and C#.
The simulation helps education and research and can allow others to do more research.
The technology is all about objects (classes - more in coding section)
The secret is an easiest way for fluid dynamics, by modelling and virtual reality. The computer does all the work by calculating physics for every individual cell. Lots of smaller calculations for each cell instead of one large calculation.
Only need to write the code once in the cell class. The program reads a file describing blood cells, and the program creates the cells each heartbeat. The program also calculates standard deviation and mean automatically.
