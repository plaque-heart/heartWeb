How to add ideas to Our Simulation which is in the "Learn" Section

Our project is Open-Source

Run a local copy of the heart site on your computer

  1. Click here so you can access our github page
  2. Use sourceTree or what ever you use to get a local copy
  3. Change to the right directory 'cd' directory
  4. Do 'pip3 install bottle'
  5. Do 'pip3 install mako'
  6. Do 'pip3 install bottle-sqlachemy'
  7. Do a localhost "python3 server.py"
  8. type in browser 'localhost:8000'
