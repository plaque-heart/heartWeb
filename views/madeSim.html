The Story Of Our Project

  1. Inspiration: First, Sophie and Kaya decided to do something on the heart when we were in Melbourne for the robogals awards weekend 2014 with our Reaction Times project
  2. Idea for the Simulation: Sophie then went 'Girls make Games' where I learnt about Unity, after that Sophie realised she could do a heart simulation
  3. A start: Sophie then went to a program she had made in the previous NCSS challenge and found a program Sophie wrote in python on around particles
    Then Sophie copied the code into a python code file and then converted to C# to make a unity program game from the program
  4. Improvements: Sophie changed the new Unity game, adding first a box, added a third dimension, and gradually changed the game until the particles moved in the way blood cells would move
  5. Our Background Script and Filming: Kaya then made a background script. Sophie and Kaya then went to The Museum of Heart Disease and filmed the background script on heart disease
  6. Blender: Sophie and Kaya then created blood cells in blender, which were imported into unity to give the correct look. Sophie and Kaya did work on creating arteries in blender, eventually finding the artery needed to be 'uwrapped' to give the correct appearance
  7. Coding physics: Then Sophie made new c# code file for the plaques, and the rest of the physics. By adding the physics the blood flowed correctly just by adding more cells to the artery for each pump of the heart
  8. Coding the Plaques: Sophie coded all the plaques to change between each other including having no plaque at all using the "p" key and the project was ready for the experiments
  9. Predictions and Movie Editing: Sophie and Kaya then made predictions for the experiments, and Sophie ran and recorded experiments and submitted a movie for Robogals Science Challenge
  10. Awesome new changes: After completing the science challenges, Sophie then made a new web server to host the project on pythonanywhere, added Virtual Reality by coding the virtual camera movement and Sophie also added a new camera and then added the code to make the camera switch
  11. Hooking the Oculus up: Sophie then bowered the Oculus from her school and a the project with Education, Experiment tools and Experiments and this web site was entered into Young ICT Explorers
  12. Presenting at young ICT Explorers(state): The project won a first prize in Young ICT explorers NSW.
  13. The Nationals at Young ICT Explorers: The project then won third prize in the Young ICT explorers National finals
  14. First ever presentation: Sophie and Kaya were asked to be presenters at an Augmented Reality Sydney Conference
  15. Robogals Weekend: The science Experiment won Second Prize at Robogals Science Challenge national awards weekend
  16. ABC3 Invatation: ABC3 invites Sophie and Kaya to be on a show called "This is Me"
